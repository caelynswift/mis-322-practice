Key Concepts

For Loop

Use when you know the range of values to iterate through. Steps from start to end one at a time.

For x As Integer = startNum To endNum ' code runs for each x Next x

Mod Operator

Returns the remainder of integer division. If x Mod 2 = 0, x is even.

4 Mod 2 ' → 0 (even) 3 Mod 2 ' → 1 (odd) If x Mod 2 = 0 Then ' x is even End If

Building Label Text in a Loop

Use & to concatenate and a space to separate values. Build the string incrementally.

evenNumbersLabel.Text = "" For x As Integer = start To endNum If x Mod 2 = 0 Then evenNumbersLabel.Text &= x & " " End If Next x

Cost Calculator
Form design, properties, Visible/Enabled, multiple button events
Concepts
Quiz

Dim variable As Type

Declare a local (procedure-level) variable using Dim.

Dim totalCost As Double totalCost = CDbl(itemCostTextBox.Text) _ * CDbl(quantityTextBox.Text) totalCostTextBox.Text = totalCost.ToString()

Visible Property

Toggle Visible to show or hide a control at runtime.

' Hide Total Cost totalCostTextBox.Visible = False totalCostLabel.Visible = False ' Show Total Cost totalCostTextBox.Visible = True totalCostLabel.Visible = True

Clear Textboxes

Use .Clear() or set .Text = "" to reset textbox content.

itemCostTextBox.Clear() quantityTextBox.Clear() totalCostTextBox.Text = ""

TextAlign & ReadOnly

Set in Properties pane. TextAlign = Right for numeric boxes. ReadOnly = True prevents user edits.

' In the Properties window: ' TextAlign → Right ' ReadOnly → True ' TabStop → False (read-only boxes)

TUT 2-1
NameYourPrice Order Form
Variables, scope (Dim vs Private), class-level vs procedure-level
Concepts
Quiz
The key challenge: deciding whether a variable should be Dim (procedure-level) or Private (class-level).

Procedure-Level (Dim)

Only needed inside ONE sub procedure. Declared inside the Sub.

Private Sub invoiceButton_Click(...) Dim subtotal As Double Dim taxAmount As Double Dim amountDue As Double ' only used here End Sub

Class-Level (Private)

Needed in MULTIPLE sub procedures. Declared at the top of the class, above all Subs.

Public Class Form1 Private quantity As Integer Private price As Double Private Sub plusOneButton_Click(...) quantity += 1 End Sub

+= and -= Operators

Shorthand to add or subtract from a variable in place.

quantity += 1 ' same as quantity = quantity + 1 quantity -= 1 ' same as quantity = quantity - 1

MyBase.Load Event

Runs when the form first opens. Good place to initialize labels (like tax rate).

Private Sub Form1_Load(...) taxLabel.Text = "0.10" End Sub

TUT 2-2
NameYourPrice (Continued)
InputBox, Option Strict On, Convert.ToDouble, .ToString formatting, constants
Concepts
Quiz

InputBox

Pops up a dialog asking the user for input. Always returns a String.

Dim priceText As String priceText = InputBox("Enter price:")

Explicit Conversion (Option Strict On)

With Option Strict On, you must explicitly convert types. Use Convert functions.

' String → Double price = Convert.ToDouble(priceText) ' String → Integer qty = Convert.ToInt32(qtyText)

.ToString Formatting

Format numbers as strings for display. "C2" = currency, "N0" = number no decimals, "N2" = 2 decimal places.

totalLabel.Text = total.ToString("C2") ' $27.50 qtyLabel.Text = qty.ToString("N0") ' 5 rateLabel.Text = rate.ToString("P1") ' 12.5%

Constants

Use Const for values that never change (like tax rates). Cannot be reassigned.

Private Const TAXRATE As Double = 0.10 ' Use TAXRATE everywhere instead ' of a variable — clearer and safer

FormatException, OverflowException, If Then traps, Exit Sub
Concepts
Quiz
Fill-in

Try / Catch / End Try

Wraps risky code. If an exception occurs the matching Catch block runs. Always end with End Try.

Try price = Convert.ToDouble(priceText) Catch fe As FormatException MessageBox.Show("Price is not a number") Exit Sub Catch oe As OverflowException MessageBox.Show("Price is too large") Exit Sub Catch ex As Exception MessageBox.Show(ex.Message) Exit Sub End Try

Order of Catch Blocks

More specific exceptions must come BEFORE the general Exception catch. VB.NET processes them in order.

' CORRECT order: Catch fe As FormatException ' ← specific Catch oe As OverflowException ' ← specific Catch ex As Exception ' ← general last

Logic Error Traps (If Then)

Use If...Then for logic errors (blank fields, negative values) BEFORE the Try/Catch conversion.

If customerNameTextbox.Text = "" Then MessageBox.Show("Name is blank.") Exit Sub End If If price < 0 Then MessageBox.Show("Price must be positive.") Exit Sub End If

Quantity Range Traps

Check BEFORE updating the variable — exit sub without changing anything.

' +1 button If quantity + 1 >= 10 Then MessageBox.Show("Max is 10") Exit Sub End If quantity += 1 ' -1 button If quantity - 1 < 0 Then MessageBox.Show("Min is 0") Exit Sub End If quantity -= 1

LAP 2
Shipping Cost App
Full app: constants, +1/-1 quantity, Try/Catch, currency formatting
Concepts
Quiz
Check figures: Goods Price = 5, Shipping = 2, Qty = 4 → Tax = $2.20, Total = $24.20

Tax Amount Formula

Tax is applied to goods × quantity plus shipping — the full subtotal before tax.

' Tax Amount = (GoodsPrice × Qty + Shipping) × TaxRate taxAmount = (goodsPrice * quantity + shippingCost) _ * TAXRATE taxTextBox.Text = taxAmount.ToString("C2")

Total Cost Formula

' Total = GoodsPrice × Qty + Shipping + Tax totalCost = goodsPrice * quantity _ + shippingCost + taxAmount totalCostTextBox.Text = totalCost.ToString("C2")

Quantity Range (1–4)

Upper limit is 4, lower limit is 1. Initial quantity = 1.

If quantity + 1 > 4 Then MessageBox.Show("Max 4 items") Exit Sub End If quantity += 1 quantityTextBox.Text = quantity.ToString()

Reset App

Return everything to initial state: clear textboxes, reset quantity to 1.

goodsPriceTextBox.Clear() shippingCostTextBox.Clear() taxTextBox.Clear() totalCostTextBox.Clear() quantity = 1 quantityTextBox.Text = "1"

TUT 3-1
T-Shirt Order Form
Select Case, If/ElseIf/Else, Random class, vbNewLine, string assembly
Concepts
Quiz

Select Case

Cleaner than long If/ElseIf chains when testing one variable against many values.

Select Case shirtColor Case "Red" shirtPrice = 10 Case "Green" shirtPrice = 12 Case "Blue" shirtPrice = 14 Case "Pink" shirtPrice = 16 End Select

Random Class

Generates pseudo-random integers. Next(1, 5) gives 1, 2, 3, or 4 (upper bound is exclusive).

Dim rnd As New Random Dim randomNum As Integer randomNum = rnd.Next(1, 5) ' 1 to 4

vbNewLine

Inserts a line break inside a string — useful for multi-line labels.

orderTotalLabel.Text = _ customerName & "'s total:" & vbNewLine _ & orderTotal.ToString("C2")

Total Calculation

Multiply price × qty, then apply (1 + taxRate) to add tax in one step.

orderSubTotal = shirtPrice * shirtQty orderTotal = orderSubTotal * (1 + TAXRATE) orderTotalLabel.Text = _ orderTotal.ToString("C2")

.ToUpper() for String Comparison

Convert both strings to uppercase before comparing to make it case-insensitive.

If string1.ToUpper() = string2.ToUpper() Then compareResult = string1 & vbNewLine _ & "is equal to" & vbNewLine & string2 Else compareResult = string1 & vbNewLine _ & "is not equal to" & vbNewLine & string2 End If

TUT 4-1
Multiple Forms
Global variables in a Module, ShowDialog(), Form_Activated event
Concepts
Quiz

Module = Global Variables

Variables in a Module are accessible from ALL forms in the project. Use Public to declare them.

Module CodeModule Public chargeAmt As Double End Module

ShowDialog vs Show

ShowDialog() opens the form modally — user cannot switch to the previous form until this one closes.

' Modal (can't switch back) TotalChargeForm.ShowDialog() ' Non-modal (can switch) TotalChargeForm.Show()

Form_Activated Event

Fires every time the form becomes active — safer than Form_Load (which only fires once) when you need to update values on return.

Private Sub TotalChargeForm_Activated(...) chargeLabel.Text = _ chargeAmt.ToString("C2") End Sub

Startup Form

With multiple forms, tell VB which form to show first. Go to Project → Properties → Startup form.

' Project Properties → Application tab: ' Startup form: TicketOrderForm ' Close a form from a Return button: Me.Close()

Radio Buttons & Checkboxes
GroupBox, CheckChanged events, Enabled property, &= string accumulation
Concepts
Quiz

Radio Buttons

Only ONE radio button in a GroupBox can be selected at a time. Check with .Checked property.

If jarlRadioButton.Checked = True Then seatPremiumLevel = 5 seatType = "Jarl" Else seatType = "Viking" End If

CheckChanged Event

Fires when a checkbox is checked OR unchecked. Enable/disable corresponding textbox.

If adultCheckBox.Checked = True Then adultTextBox.Enabled = True Else adultTextBox.Enabled = False adultTextBox.Clear() End If

&= for String Accumulation

Append to a string variable without re-typing the variable name.

If adultQty <> 0 Then orderSummary = adultQty.ToString("N0") _ & " Adult " & seatType _ & " Tickets " & adultCost.ToString("C2") _ & vbNewLine End If If childQty <> 0 Then orderSummary &= childQty.ToString("N0") _ & " Child " & seatType & " Tickets " _ & childCost.ToString("C2") & vbNewLine End If
